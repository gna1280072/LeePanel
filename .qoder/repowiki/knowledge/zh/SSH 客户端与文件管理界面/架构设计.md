- 采用 React 函数组件与 Hooks 构建 UI，通过 `@tauri-apps/api` 的 `invoke` 和 `listen` 与 Rust 后端进行双向通信（命令调用与事件订阅）。
- 核心布局由 `App.tsx` 统筹，集成可拖拽调节宽度的侧边栏（`Sidebar`）、连接表单（`ConnectForm`）以及上下分屏的文件浏览器（`FileBrowser`）和终端（`Terminal`）。
- 终端模块封装 `@xterm/xterm`，通过 `forwardRef` 暴露命令发送接口，并监听后端推送的 SSH 输出流实现实时渲染。
- 文件浏览器实现复杂的本地状态管理，包括右键上下文菜单、拖拽排序/移动、内置文本编辑器模态框以及下载进度追踪。
