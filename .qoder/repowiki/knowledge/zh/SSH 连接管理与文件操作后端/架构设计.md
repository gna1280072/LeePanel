- 入口与生命周期：`src/main.rs` 仅调用 `app_lib::run()`；核心逻辑在 `src/lib.rs` 的 `run()` 中构建 Tauri 应用并注册命令。
- 状态管理：通过 `tauri::State<Arc<Mutex<SshManager>>>` 共享 SSH 会话管理器，保证并发安全与会话隔离。
- 模块划分：`ssh.rs` 负责 SSH/SFTP 会话建立、通道复用、文件读写与进度事件发射；`config.rs` 负责连接配置与应用设置的 JSON 持久化（位于系统配置目录）。
- 通信边界：前端通过 `#[tauri::command]` 调用后端；后端通过 `AppHandle::emit` 向窗口发送 `ssh-output`、`upload-progress`、`download-progress` 等事件。
- 依赖方向：UI → Tauri Invoke Handler → SshManager/ConfigManager → russh/russh-sftp/文件系统。
