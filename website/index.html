免费开源，下一代 Linux 服务器管理面板。

目前流行的各种面板需要安装在服务器上，而这些面板自身经常发现安全问题，令广大的服务器管理员苦不堪言。

我们开发 LeePanel，希望彻底解决这个问题。

所有操作通过本地电脑发送 SSH 命令执行，服务器上不安装任何面板代码——从根本上更安全。

基于 Tauri 2 + React 构建的轻量级跨平台桌面应用。通过单一原生客户端统一管理 SSH、文件、Nginx、MySQL、PHP、Redis、Docker、防火墙、SSL 等功能。

项目发布于 2026 年 7 月 18 日，仍处于早期阶段。当前已在 Ubuntu/Debian 主流版本测试通过，更多 Linux 发行版适配中。

有任何建议或反馈？欢迎到 GitHub Discussions 提出。

零服务器代码

不在服务器部署任何面板程序，所有指令通过 SSH 远程执行，从根源上消除面板漏洞风险。

🖥️

跨平台桌面应用

基于 Tauri 2 构建，原生支持 Windows、macOS 和 Linux。告别浏览器标签页，享受原生桌面体验。

🆓

免费开源

完全免费，源代码公开透明。社区驱动，持续迭代，没有隐藏收费，没有功能限制。

功能特性

一个客户端，搞定所有服务器管理需求

⌨️

SSH 终端

密码/密钥认证，xterm.js 全功能终端，支持剪贴板、链接识别、自动重连。

📁

文件管理

远程文件浏览器，拖拽上传下载，分片上传大文件，压缩解压，权限管理。

🔧

LNMP 一键安装

Nginx、MySQL/MariaDB、PHP-FPM 一键安装，版本可选，服务启停控制。

🌐

站点管理

Nginx 虚拟主机、Let's Encrypt SSL、反向代理、伪静态、PHP 版本切换。

🗄️

数据库管理

MySQL/MariaDB 数据库 CRUD、用户权限、备份恢复、SQL 导入、Root 密码管理。

⚡

Redis 管理

Key 浏览与 CRUD、TTL 支持、多数据库切换、备份恢复。

🐳

Docker 管理

容器生命周期管理、镜像拉取运行、日志查看、镜像源配置。

📊

系统监控

实时 CPU、内存、磁盘、网络监控，进程列表，运行时间追踪。

🛡️

防火墙 & BBR

ufw/firewalld/iptables 规则管理，BBR TCP 拥塞控制一键开启。

🔑

SSL 证书

Let's Encrypt 证书自动申请、续期、部署，支持多域名和泛域名。
