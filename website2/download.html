System requirements

  • Windows — 10 or later, 64-bit
  • macOS — 11 or later, Intel & Apple Silicon
  • Linux — any glibc distro (.deb tested on Ubuntu / Debian)
  • Servers — any Linux box you can reach over SSH; nothing is ever installed on them
  • Network — outbound SSH (port 22 by default). That's it.

Built-in updates

LeePanel checks this exact feed on launch and asks you before downloading anything — no silent installs, no background daemon living between checks.

Verify your build

Windows and Linux installers carry a signature pinned in the manifest. The updater refuses any build whose signature doesn't match the embedded public key.
