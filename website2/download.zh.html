系统要求

  • Windows — 10 及以上，64 位
  • macOS — 11 及以上，Intel 与 Apple Silicon
  • Linux — 任意 glibc 发行版（.deb 已在 Ubuntu / Debian 验证）
  • 服务器 — 任何能通过 SSH 连上的 Linux 机器，不装任何东西
  • 网络 — 只需出站 SSH（默认 22 端口），别无其他

内置更新

LeePanel 启动时检查的正是这个数据源，下载任何东西之前都会先问你——没有静默安装，也没有常驻后台的守护进程。

校验你的安装包

Windows 与 Linux 安装包携带清单中固定的签名。如果签名与内置公钥不匹配，更新器会拒绝安装。
